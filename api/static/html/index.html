USERS

Http Methods Path Description
POST /login User login.
GET /user/users Display a list of users.
GET /user/user/:id Display a single user.
PUT /user/user/:id Update user's record.
PATCH /user/user/:id To modify a user's record.
POST /user/register To register a new user.
DELETE /user/user/:id To delete a single user.

PRODUCTS

Http Methods Path Description
GET /product/products Display all products.
GET /product/product/:id Display a single product by ID.
PUT /product/product/:id To update a product info.
POST /product/addProduct To add a new product.
DELETE /product/product/:id To delete a product.

CATEGORIES

Http Methods Path Description
GET /category/categories Display a list of all categories.
POST /category/addCategory To add a new category.
DELETE /category/category/:id To delete a category.

SUPPLIERS

Http Methods Path Description
GET /supplier/suppliers Display a list of suppliers.
GET /supplier/supplier/:id Display a single supplier.
PUT /supplier/supplier/:id Update supplier record.
POST /supplier/addSupplier To add a new supplier.
DELETE /supplier/supplier/:id To delete a supplier.

ORDERS

Http Methods Path Description
GET /user/:id/orders Display all users orders.
PUT /user/:id/order/:id Update users orders.
POST /user/:id/order To add a new order to users account.
DELETE /user/:id/order To delete all orders from user account.
DELETE /user/:id/order/:id To delete a single order from users acount.

PAYMENT

Http Methods Path Description
GET /user/:id/payments Display all users orders made.
PUT /user/:id/payment/:id Update users orders, history.
POST /user/:id/payment To add a new completed payment to users account.
DELETE /user/:id/payment To delete all history of payments from user account.
DELETE /user/:id/payment/:id To delete a single order history from users acount.
